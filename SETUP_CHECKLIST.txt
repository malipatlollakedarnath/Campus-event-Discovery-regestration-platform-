🚀 QUICK START (5 MINUTES)
================================

1. OPEN QUICKSTART.html
   - Open: ./QUICKSTART.html in your browser
   - Follow the 8-step interactive guide
   - It walks you through the entire setup

2. GET FIREBASE CONFIG
   - Go to firebase.google.com
   - Create project → Enable Firestore & Auth
   - Copy config to js/firebase-config.js

3. RUN THE APP
   - Open index.html in browser
   - Or use: python -m http.server 8000
   - Visit: http://localhost:8000

4. TEST THE APP
   - Create student account
   - Create admin account (use @admin email)
   - Create sample event
   - Register as student

🐛 COMMON ISSUES & FIXES
================================

Issue: "Firebase config not found" error
Fix: Update js/firebase-config.js with your Firebase credentials

Issue: "Functions don't load"
Fix: Check browser console (F12) for errors

Issue: Can't login
Fix: Make sure Firestore Database exists and Auth is enabled

Issue: Events not showing
Fix: Check internet connection and Firestore rules

Issue: Mobile looks broken
Fix: Refresh page, CSS should adapt automatically
