🚀 Quick Start Guide

Step 1: Create Firebase Project

  1. Go to firebase.google.com
  2. Click "Get started"
  3. Create a new project called campus-events
  4. Accept all default settings
  5. Click "Create project"
  6. Wait for project to be created (1-2 minutes)

Step 2: Enable Firestore Database

  1. In Firebase Console, click "Firestore Database" from left menu
  2. Click "Create Database"
  3. Choose "Start in test mode" (for development)
  4. Select closest region to you
  5. Click "Create"
  6. Wait for database to be created

Step 3: Enable Authentication

  1. Go to "Authentication" from left menu
  2. Click "Get started"
  3. Find "Email/Password" and click it
  4. Toggle "Enable" to turn it on
  5. Click "Save"

Step 4: Get Firebase Configuration

  1. In Firebase Console, click the ⚙️ Settings icon → "Project settings"
  2. Go to "Your apps" section
  3. Click the web icon </>
  4. Register your app (give it any name)
  5. Copy the entire firebaseConfig object from the code shown
  6. Example format: 
    {
  apiKey: "AIzaSyDxxxxxxxxxx",
  authDomain: "campus-events.firebaseapp.com",
  projectId: "campus-events",
  storageBucket: "campus-events.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
}

Step 5: Update firebase-config.js

  1. Open the file: js/firebase-config.js
  2. Find this section: 
    const firebaseConfig = {
    apiKey: "AIzaSyDxxxxxxxxxxxxxxxxxxxxxxxxx",
    authDomain: "campus-events-xxxxx.firebaseapp.com",
    projectId: "campus-events-xxxxx",
    storageBucket: "campus-events-xxxxx.appspot.com",
    messagingSenderId: "xxxxxxxxx",
    appId: "1:xxxxxxxxx:web:xxxxxxxxxxxxxxx"
};
  3. Replace it with your config from Step 4
  4. Save the file

Step 6: Add Firestore Rules (Optional but Recommended)

  1. In Firebase Console, go to "Firestore Database" → "Rules" tab
  2. Replace all text with rules from FIRESTORE_RULES.txt
  3. Click "Publish"
  4. If you skip this, use test mode rules (less secure)

Step 7: Run the Application

  1. Open index.html in your browser
  2. Or use a local server:
    • Python 3: python -m http.server 8000
    • Node.js: npx http-server
    • VS Code: Install Live Server extension → Right-click index.html → Open with Live Server
  3. Visit: http://localhost:8000

Step 8: Test the Application

  3. Enter:

    • Email: student@example.com

  2. Enter:

  2. Fill in:

    • Description: "Learn about new technologies"

  1. Log in as the student
  2. See the "Tech Talk 2026" event
  3. Click "Register"
  4. Check "My Registrations" to see it

✅ Verification Checklist

  • Firebase project created
  • Firestore Database enabled
  • Authentication enabled (Email/Password)
  • Firebase config updated in firebase-config.js
  • Application runs without errors
  • Students can register and login
  • Admins can create events
  • Students can see and register for events
  • Registrations appear in admin panel

🐛 Common Issues

Q: "undefined is not a function" error?

A: Check that firebase-config.js has your Firebase credentials. Look at browser console (F12).

Q: Login button does nothing?

A: Make sure Authentication is enabled and Email/Password provider is turned on.

Q: Events don't appear?

A: Check Firestore Database is created and test mode rules are applied.

Q: Can't register as admin?

A: Use email with @admin or admin@college domain (e.g., admin@admin.com).

Q: Still having issues?

A: Check browser console (F12) for error messages. Review README.md for more details.
